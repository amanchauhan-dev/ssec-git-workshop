# SSEC Git & GitHub Workshop

This repository contains materials, examples, and hands-on exercises for the **Git & GitHub Workshop** conducted at **Shantilal Shah Engineering College (SSEC)**.

The goal of this workshop is to help students learn **version control**, **collaboration workflows**, and **industry-level Git practices** from beginner to advanced levels.

---

## 📌 What You Will Learn

This workshop covers:

- Introduction to Version Control
- Git Fundamentals
- GitHub Collaboration
- Branching and Merging
- Pull Requests and Code Review
- Git Rebase and Merge Strategies
- Git Stash and Undo Commands
- Cherry-Pick and Bisect
- Git Tags and Releases
- GitHub Actions (CI/CD Basics)
- Team Collaboration Workflow

---

## 🛠 Hands-On Topics

Participants will practice:

- Creating repositories
- Tracking file changes
- Working with branches
- Resolving merge conflicts
- Using pull requests
- Managing project history
- Collaborating in teams
- Simulating real-world workflows
